#  To-Do List Web Application

## Project Overview
Welcome to our simple To-Do List web application built as a **hackathon group project**. The application allows users to add, view, mark as complete, and delete tasks with a modern interface.

The project demonstrates collaborative web development with team members focusing on different aspects of the stack (HTML, CSS, and JavaScript), while working together to create a polished final product.

---

## Features
-  Add new tasks dynamically
-  Mark tasks as completed (click on task)
-  Delete individual tasks
-  Pre-loaded example tasks
-  Modern gradient UI with smooth animations
-  Input validation (prevents empty tasks)
-  Keyboard support (Enter key to add tasks)

---

##  Technologies Used
- **HTML5** - Semantic page structure
- **CSS3** - Modern styling with gradients, flexbox, and animations
- **JavaScript (ES6)** - Dynamic DOM manipulation and event handling

## How to Use

1. **Clone or download** this repository
2. **Open `index.html`** in any modern web browser
3. **Add tasks:**
   - Type a task in the input field
   - Click "Add Task" or press Enter
4. **Complete tasks:**
   - Click on any task to mark it as completed (strikethrough)
   - Click again to unmark
5. **Delete tasks:**
   - Click the "Delete" button on any task to remove it
